# 需求描述
想做一个自动化基线测试项目，包含：构建、数据集管理、运行、评估、回归对比、存储加可视化、报告、CI/CD集成等流程。

1. 构建：目前在本地环境运行。
2. 数据集管理：本地数据集。
3. 运行：本地运行。
4. 评估：本地评估。
5. 回归对比：本地回归对比。
6. 存储加可视化：持久化评估结果，用于回归对比。并根据需求提供可视化。
7. 报告：生成报告，用于展示评估结果、回归对比结果。

未来可以考虑将以上流程进行CI/CD集成，独立于本地环境。

# 难点
1. 目前只需要支持 一两个特定算法 的自动化基线回归测试。 但后期可能要扩展至其他定位算法，因此必须考虑可扩展性。
2. 不同算法构建方式不同；
3. 不同算法运行时需要传入的参数不同；
4. 不同算法评估方式不同；
5. 不同算法可视化内容、报告内容也不同；
